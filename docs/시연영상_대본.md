# 🎬 N분의1 — 시연영상 대본 (화면 녹화용)

> 팀 프로젝트 기여도 증명·관리 플랫폼 · 앱 내 **모든 기능** 시연
> 구성: **[화면] → [조작] → 🎙️나레이션** · 예상 길이 **약 4~5분**

---

## ✅ 촬영 전 준비
- 기기: 안드로이드 폰(또는 에뮬레이터), 화면 녹화 ON, 알림 OFF, 밝기 최대
- **팁: 대부분 로그인 화면의 `🎬 데모 데이터로 둘러보기`로 시작하면 데이터가 채워진 화면으로 시연할 수 있어요.** (입력 기능 시연은 그 안에서 직접 눌러 보여주면 됩니다)
- 흐름: **① 로그인 → ② 학생 경험 → ③ 팀 만들기 → ④ 교수 경험 → ⑤ 설정 → ⑥ 마무리**

## 0. 오프닝 (앱 아이콘 → 로그인 화면)
**[화면]** 앱 실행 → 로그인 화면
🎙️ "안녕하세요, 팀 N분의1입니다. 저희 앱은 팀 프로젝트의 기여도를 **기록하고 증명하며, 교수님이 근거로 평가**하는 플랫폼입니다. 지금부터 실제 화면으로 기능을 하나씩 보여드리겠습니다."

## 1. 로그인 & 온보딩
**[화면]** 로그인 화면 (1/N 로고, 학생·교수님 카드, 카카오·구글 버튼, 데모 버튼)
**[조작]** 학생 카드 탭 → 이름 입력란에 이름 타이핑 → `카카오로 시작하기`(또는 Google) 버튼을 손가락으로 가리키기
🎙️ "먼저 로그인입니다. **학생과 교수님** 두 가지 역할이 있고, **카카오·구글 간편 로그인**을 지원합니다. 학생은 팀에서 쓸 이름만 입력하면 바로 시작할 수 있습니다."
**[조작]** `🎬 데모 데이터로 둘러보기` 버튼 탭
🎙️ "심사·시연을 위해 샘플 데이터가 채워진 상태로 둘러보겠습니다. 먼저 **학생 화면**입니다."

## 2. 학생 — 홈 화면 (개인 대시보드)
**[화면]** 학생 홈 (안녕하세요 OOO님, 🔥 스트릭, 다가오는 마감, 빠른 실행, 내 팀, 기여도 잔디)
**[조작]** 화면을 천천히 위→아래로 스크롤
🎙️ "학생 홈입니다. 상단엔 **연속 기록 스트릭**이 있어 매일 참여를 유도하고, 그 아래 **다가오는 마감**으로 팀 프로젝트 일정을 놓치지 않게 해줍니다."
🎙️ "**빠른 실행**으로 활동 기록·동료평가·산출물·팀 추가에 바로 접근하고, **내 팀** 카드에서 팀원 수·활동 건수·평가 현황을 확인합니다."
🎙️ "맨 아래 **기여도 잔디**는 깃허브 잔디처럼 제 활동을 시각화해, '얼마나 꾸준히 기여했는지'를 한눈에 보여줍니다."

## 3. 팀 상세 — 5개 탭 전체
**[조작]** `내 팀`의 팀 카드(N분의1) 탭 → 팀 상세 진입
🎙️ "팀을 열면 **개요·일정·활동·평가·산출물** 다섯 개 탭으로 팀플의 모든 것을 관리합니다."

### 3-1. 개요 탭
**[화면]** 개요 (팀원 및 역할, 활동 현황 차트, GitHub, 팀 설문)
**[조작]** 스크롤하며 팀원 카드 → 활동 현황 막대 차트 보여주기
🎙️ "개요 탭입니다. **팀원과 역할**, 그리고 팀원별 **활동 시간 막대 차트**로 누가 얼마나 기여했는지 팀원끼리도 투명하게 공유됩니다. `+ 팀원 추가`로 팀원을 초대할 수 있고요."
**[조작]** `GitHub 저장소` 접이식 헤더 탭 → 펼치기
🎙️ "**깃허브 저장소**를 등록하면 교수님 화면에서 커밋 기여까지 분석됩니다."
**[조작]** `팀 설문` 접이식 헤더 탭 → 펼치기 → 설문 결과 막대그래프 → `에브리타임에 공유` 버튼 탭
🎙️ "**팀 설문** 기능으로 설문을 만들고 **에브리타임에 바로 공유**해 응답을 모읍니다. 프로젝트 근거 데이터로 활용할 수 있습니다."
**[조작]** 맨 아래 `팀 관리` → `팀 정보 수정` 잠깐 보여주기
🎙️ "그리고 **팀 이름·프로젝트명 수정, 팀원 삭제, 팀 삭제**까지 팀 관리도 자유롭게 됩니다."

### 3-2. 일정 탭 (미니 LMS 연동)
**[조작]** `일정` 탭 → 마일스톤 목록 스크롤
🎙️ "일정 탭입니다. 과목에 연결하면 **제안서·중간발표·최종발표** 같은 마일스톤이 표시되고, **D-day와 마감 임박**이 색으로 구분됩니다."
**[조작]** 예정 마일스톤의 `제출하기` 탭 → 메모 입력 + `📎 파일 첨부` 보여주기 → 제출
🎙️ "학생은 **메모와 파일을 첨부해 제출**하고, 제출하면 초록색으로 완료 표시됩니다."

### 3-3. 활동 탭 (핵심 입력)
**[조작]** `활동` 탭 → 카테고리 칩(개발 등) 선택 → '무엇을 했나요?' 입력 → 소요시간 슬라이더 조절 → 분위기 이모지 하나 탭 → `기록 남기기`
🎙️ "가장 핵심인 활동 기록입니다. **카테고리, 한 일, 소요 시간**을 남기고, **오늘의 팀플 기분**을 이모지로 체크인합니다. 이 기분 데이터는 교수님이 팀 분위기나 번아웃을 조기에 감지하는 데 쓰입니다."
🎙️ "기록하면 아래 목록에 바로 쌓이고, '기록 완료' 알림과 함께 잔디가 채워집니다. **말이 아니라 기록으로 증명**하는 거죠."

### 3-4. 평가 탭 (익명 동료평가)
**[조작]** `평가` 탭 → 팀원별 **기여도·책임감·협업·소통** 슬라이더 조절 → 한 줄 코멘트 입력 → `평가 제출하기`
🎙️ "평가 탭에서는 팀원을 **기여도·책임감·협업·소통 네 항목**으로 평가하고 한 줄 코멘트를 남깁니다. **완전 익명으로 집계**되어 솔직한 평가가 가능하고, 이게 나중에 AI 교차검증의 두 번째 근거가 됩니다."

### 3-5. 산출물 탭
**[조작]** `산출물` 탭 → `파일 선택하기` → (발표자료·코드 등) 첨부 → 목록에 표시
🎙️ "산출물 탭에서는 발표자료·코드·보고서 같은 결과물을 첨부해 **기여의 물증**을 남깁니다."

## 4. 팀 만들기 흐름 (신규 생성)
**[조작]** 뒤로가기 → 홈 → `빠른 실행`의 `팀 추가`(또는 `팀 만들기`) 탭 → 팀 이름·프로젝트명·설명 입력 → 팀원 이름·역할 추가 → `팀 만들기`
🎙️ "새 팀은 이렇게 만듭니다. **팀 이름과 프로젝트, 팀원과 역할**을 입력하면 되고, 만들자마자 팀 상세로 바로 들어가 기록을 시작할 수 있습니다."

## 5. 교수 — 대시보드
**[조작]** 우측 상단 로그아웃 → 로그인 화면 → `교수님` 카드 선택 → `🎬 데모 데이터로 둘러보기`
🎙️ "이번엔 **교수님 화면**입니다."
**[화면]** 교수님 대시보드 (담당 팀·확인 필요·평균 건강도 요약, 이번 주 마감, 내 과목, 담당 팀)
**[조작]** 상단 요약 3개 → 이번 주 마감 배너 → 담당 팀 목록 스크롤(확인 필요 뱃지 강조)
🎙️ "교수님은 **담당 팀 수, 확인이 필요한 팀 수, 평균 팀 건강도**를 한눈에 봅니다. **이번 주 마감**이 알림처럼 뜨고, 담당 팀은 **'관심이 필요한 팀' 순서**로 정렬되어 문제 팀부터 바로 확인할 수 있습니다."

## 6. 교수 — 미니 LMS (과목 개설·관리)
**[조작]** `+ 과목 개설` 탭 → 과목명·학기 입력 → 유형(졸업작품/중간·기말/일반) 선택 → 자동 생성 마일스톤 미리보기 → `과목 개설하기`
🎙️ "교수님은 과목을 개설할 수 있습니다. **유형만 고르면** 졸업작품·중간기말 프로젝트에 맞는 **마일스톤 일정이 자동 생성**됩니다. 졸업작품 관리, 중간·기말 프로젝트 관리를 이 앱 하나로 하는 거죠."
**[조작]** 과목 상세 → 팀×마일스톤 **진행 매트릭스** 보여주기 → 마일스톤 `✏️ 수정`/삭제 → 아래 `과목 관리`의 `과목 정보 수정`·`과목 삭제`
🎙️ "과목 안에서는 **팀별 마일스톤 진행 현황이 색 매트릭스**로 보이고, 마일스톤과 과목 모두 **수정·삭제**가 자유롭습니다."

## 7. 교수 — 팀 분석 (킬러 기능) ⭐
**[조작]** 대시보드로 돌아가 `확인 필요` 뱃지가 있는 팀(예: 캡스톤A) 탭
🎙️ "이제 저희 앱의 핵심입니다."
**[화면]** 교차검증 플래그 카드들 (불일치/기록 누락/무임승차)
**[조작]** 플래그 카드 천천히 보여주기
🎙️ "**교차검증 플래그**입니다. 본인 기록과 동료평가를 대조해, **기록은 많은데 평가가 낮으면 '불일치', 반대면 '기록 누락 가능성', 둘 다 낮으면 '무임승차 의심'**을 **앱이 먼저 자동으로** 잡아 빨강·노랑 카드로 알려줍니다."
**[조작]** `참고 기여도` 차트 → `복사` 버튼 탭 → `활동 시간 분포` 도넛 차트
🎙️ "그 아래 **참고 기여도**는 활동 60%와 동료평가 40%를 종합한 수치로, **복사 버튼으로 성적 입력에 바로** 쓸 수 있습니다. 활동 시간 분포도 도넛 차트로 보여주고요. 다만 **확정 점수가 아니라 참고 자료이며, 최종 판단은 교수님 몫**으로 남깁니다."
**[조작]** `AI 기여도 리포트` 생성 버튼 탭 → 리포트 생성 → `PDF로 공유` 보여주기
🎙️ "마지막으로 **AI 기여도 리포트**입니다. Claude AI가 모든 기록과 평가를 종합해 **팀별 기여도 근거 문서를 자동 생성**하고, **PDF로 내보내** 성적 근거로 보관할 수 있습니다."

- [ ] 활동: 카테고리·소요시간·**분위기 이모지 체크인**·기록
- [ ] 평가: **4항목 익명 동료평가**·코멘트
- [ ] 산출물: **파일 업로드**
- [ ] 팀 만들기(생성 흐름)
- [ ] 교수 대시보드: 요약·**이번 주 마감**·내 과목·**관심 필요 순 정렬**
- [ ] 미니 LMS: **과목 개설(유형별 자동 마일스톤)**, 진행 매트릭스, 마일스톤 **수정/삭제**, 과목 **수정/삭제**

- [ ] **AI 기여도 리포트 생성 + PDF 공유**
- [ ] 설정: **다크모드 + 5색 테마**, API 키 암호화
